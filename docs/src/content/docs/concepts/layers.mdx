## Loading Strategy

| Tier | Contents | When loaded |
|---|---|---|
| **Hot** | nrs · corporate · team · project | Always in agent memory |
| **Warm** | domain · implementation | On entering directory |
| **Cold** | `docs/` | On demand when task requires |

## The Project Map

`project.context.md` is the entry point. It indexes: purpose, architecture, domains, documentation (as markdown links), tools, commands, skills, MCP servers. From this file, an agent can find everything else.
